Instructions

You must solve a minesweeper game with 16 rows and 30 columns with 99 bombs.

You can use the following commands

  • To open a cell : http://<server-address>/open?id=<your id>&x=<x (column no starting on 0)>&y=<y (row)>
  • Let the server open a cell for you - http://<server-address>/open?id=<your id>&x=<x (column no starting on 0)>&y=<y (row)>
  • Check how you board looks like now (used for debug) http://<server-address>/debugdisplay?id=<your id>

You will receive an answer like this:

'Y=3,X=5,result=4'

This indicates that the sixth column on the fourth row has four bombs. The cell in the upper left corner has coordinate 0,0

If you get a result like :bomb or :open it means that you have opened a bomb or an already open field. Your score will then be set to 0 and a new board will be generated for you to start over

You get points for each finished board. Fewer hints used means more points. If you open a mine or a cell that is already opened your score will be set to zero

When you finish a board (or a mine blows up) a new board will be generated automatically
